REST API ContribsGH-P-C

A REST service that given the name of a GitHub organization returns a list of contributors (by organization or by repo) in JSON format, sorted by the number of contributions

Parallel version using Scala futures with a cache

How to use the service

  1. Clone the repository at https://github.com/avargasv/ContribsGH-P-C
  2. Run sbt under the repository's home directory
  3. After the sbt prompt 'sbt:ContribsGH-P-C>' appears, execute the following sbt command:
    jetty:start
  4. Call the rest service with the following url:
    http://localhost:8080/org/{org_name}/contributors?group-level={organization|repo}&min-contribs={integer_value}
    or play with the service using the page provided at http://localhost:8080/index

Notes on the implementation

  • The service uses GitHub REST API v3
  • The JSON format of the contributors returned is:
    { “repo”: <repo_name>, “contributor”: <contributor_login>, “contributions”: <no_of_contributions> }
  • If the environment variable GH_TOKEN is set, the service uses its value for authentication using a request header
